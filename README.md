# SimplePong

Original tutorial can be found under "Simple 2D game" at Godot Docs
https://godot.readthedocs.io/en/stable/learning/step_by_step/simple_2d_game.html

As a learning exercise, a few things were added:
- scoreboard
- main menu
- one player vs. AI
- instructions
